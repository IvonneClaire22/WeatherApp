# Weather App

A simple, interactive weather app that fetches current weather data using the OpenWeatherMap API.

## Features
- Search for any city
- Displays temperature, humidity, and wind speed
- Weather icon changes based on weather conditions
- Error message shown for invalid city names

## How to Use
1. Open `index.html` in your browser.
2. Enter a city name in the input field.
3. Click the search button to view weather information.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
